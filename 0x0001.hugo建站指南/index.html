0x0001.Hugo建站指南

七月 16, 2025

Hugo建站教程：从安装到部署#

1. 安装Hugo#

Windows/macOS/Linux通用方法：

# macOS (需先安装Homebrew)
brew install hugo

# Windows (需要先安装Chocolatey)
choco install hugo-extended

# Linux (Debian/Ubuntu)
wget https://github.com/gohugoio/hugo/releases/download/v0.125.0/hugo_extended_0.125.0_Linux-64bit.deb
sudo dpkg -i hugo_extended_*.deb
#Linux（备选，需安装Snap）
sudo snap install hugo --channel=extended

验证安装：

hugo version
# 输出示例: hugo v0.125.0+extended

2. 新建网站#

# 创建新站点（默认生成config.toml）
hugo new site my-blog

# 可选：生成YAML格式配置（推荐）
hugo new site my-blog --format yaml

两种配置文件格式对比：

格式 示例 特点
TOML baseURL = "https://..." 类INI语法
YAML baseURL: "https://..." 缩进敏感，易读

2.1 配置网站#

编辑hugo.yaml（参考你的配置文件）：

baseURL: ""                     # 部署后替换为真实域名
defaultContentLanguage: zh      #设置默认语言为中文
languageCode: zh-CN             #设置语言为简体中文
hasCJKLanguage: true
title: ""
theme: PaperMod

params:
  defaultTheme: auto            # 自动切换深色/浅色模式
  disableCookieBanner: true     # 禁用烦人的Cookie横幅

menu:
  main:
    - identifier: home
      name: "首页"
      url: "/"
      weight: 1

2.2 安装主题（以PaperMod为例）#

方法1.使用Git Clone

git clone https://github.com/adityatelange/hugo-PaperMod themes/PaperMod --depth=1

方法2.使用Git Submodule

git submodule add --depth=1 https://github.com/adityatelange/hugo-PaperMod.git themes/PaperMod
git submodule update --init --recursive

创建自定义CSS（可选）：

  1. 新建文件 assets/css/custom.css
  2. 在配置中添加：
params:
  assets:
    customCSS:
      - css/custom.css

3. 部署站点#

3.1 GitHub Pages#

步骤：

  1. 创建仓库 username.github.io

  2. 在本地设置仓库：

    cd public
git init
git branch -M main
git remote add origin https://github.com/OWNER/REPOSITORY.git

  3. 将public目录推送到仓库的main分支

    git add .
git commit -m "此处填写commit信息"
git push

  4. 进入仓库设置 > Pages > 选择分支

    注意：需在配置中设置：

baseURL: "https://username.github.io"
3.2 CloudFlare Pages#

  1. 将代码推送到Git仓库（GitHub/GitLab）

  2. 登录CloudFlare控制台 > Pages > 创建项目

  3. 构建设置：

    • 构建命令：不填
    • 输出目录：不填
    • 环境变量：不填
3.3 Netlify#
  1. 推送代码到Git仓库
  2. 登录Netlify > Add new site > Import project
  3. 构建设置：
    • Build command: 不填
    • Publish directory: 不填

4. 常用命令速查#

命令 作用
hugo new posts/1.md 新建文章
hugo server -D 启动本地服务（含草稿）
hugo --minify 生成优化后的静态文件
hugo --theme PaperMod 指定主题构建

部署成功后，可通过你的域名访问网站。 完整的原始教程参考：Hugo文档
