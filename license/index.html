第三章 侵权责任

3.1 分级处理机制

侵权类型处理方式法律依据
普通侵权（非主观故意）收到通知后7日内整改可免予追究《民法典》第一千一百六十五条
商业侵权（含身份营销）立即适用3.2条赔偿责任《著作权法》第五十四条

3.2 赔偿计算标准

最终赔偿额 = 赔偿基数 × 3 + 维权合理支出

计算细则：

  1. 赔偿基数按以下顺序确定：
    • 优先采用侵权方实际获利金额（需提供财务凭证）
    • 无法确定时采用法定保底标准：5000元/件
  2. 维权合理支出包括：律师费、公证费、差旅费等直接费用

计算示例：

  • 侵权方获利1000元 → 赔偿额 = 1000×3 + 维权支出
  • 无法确定获利 → 赔偿额 = 5000×3 + 维权支出

3.3 维权程序

  1. 侵权通知
    向侵权方或传播平台发送《整改/删除告知书》
  2. 证据保全
    必要时采用区块链存证（根据《电子签名法》第七条）
  3. 司法追偿
    逾期未整改的，向侵权行为地或被告住所地人民法院提起诉讼（依据《中华人民共和国民事诉讼法》第二十四条）
